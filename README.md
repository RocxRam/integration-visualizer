# 📊 Numerical Integration Visualizer

A Python-based interactive tool to visualize and compare numerical integration methods:
- Riemann Sum
- Midpoint Rule
- Trapezoidal Rule

This project helps understand how numerical integration approximates definite integrals using graphical interpretation.

---

## ✨ Features

- Accepts mathematical functions using SymPy
- Computes numerical integrals using 3 methods
- Visualizes area approximation using Matplotlib
- Comparison mode (side-by-side graphs)
- Handles improper inputs and invalid expressions

---

## 📐 Methods Implemented

### 🔹 Riemann Sum
Approximates area using rectangles at interval endpoints.

### 🔹 Midpoint Rule
Uses midpoint of each interval for better accuracy.

### 🔹 Trapezoidal Rule
Approximates area using trapezoids between points.
